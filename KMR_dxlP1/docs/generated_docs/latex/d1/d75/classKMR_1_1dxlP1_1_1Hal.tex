\doxysubsection{Detailed Description}
Hardware abstraction layer for Dynamixel motors. 

The lowest-\/level element in the library. The \doxylink{classKMR_1_1dxlP1_1_1Hal}{Hal} class serves primarily as an abstraction layer, providing high-\/level functions to get the Dynamixel control table addresses by creating a control table. ~\newline
 It also parses the project\textquotesingle{}s motors configuration file.
