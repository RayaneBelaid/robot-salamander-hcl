\doxysubsection{Detailed Description}
Parent class, to be specialized as a \doxylink{classKMR_1_1dxlP1_1_1Reader}{Reader} or \doxylink{classKMR_1_1dxlP1_1_1Writer}{Writer}. 

This class is not usable by itself, it is a non-\/specialized sketelon inherited by the child classes \doxylink{classKMR_1_1dxlP1_1_1Reader}{Reader} and \doxylink{classKMR_1_1dxlP1_1_1Writer}{Writer}. ~\newline
 It contains functionalities to check the viability of sync readers/writers that will be defined in child classes (motor compatibility).
