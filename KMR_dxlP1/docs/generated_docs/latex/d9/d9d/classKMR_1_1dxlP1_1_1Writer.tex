\doxysubsection{Detailed Description}
Custom \doxylink{classKMR_1_1dxlP1_1_1Writer}{Writer} class that contains a dynamixel\+::\+Group\+Sync\+Write object. 

This custom \doxylink{classKMR_1_1dxlP1_1_1Writer}{Writer} class simplifies greatly the creation of dynamixel writing handlers. ~\newline
 It takes care automatically of address assignment, even for indirect address handling.
