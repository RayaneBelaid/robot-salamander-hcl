\doxysubsection{Detailed Description}
Custom \doxylink{classKMR_1_1dxlP1_1_1Reader}{Reader} class that contains a dynamixel\+::\+Group\+Sync\+Read object. 

This custom \doxylink{classKMR_1_1dxlP1_1_1Reader}{Reader} class simplifies greatly the creation of dynamixel reading handlers. ~\newline
 It takes care automatically of address assignment, even for indirect address handling.
