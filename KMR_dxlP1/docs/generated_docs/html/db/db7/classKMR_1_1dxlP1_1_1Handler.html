Detailed Description

Parent class, to be specialized as a Reader or Writer.

This class is not usable by itself, it is a non-specialized sketelon inherited by the child classes Reader and Writer.
It contains functionalities to check the viability of sync readers/writers that will be defined in child classes (motor compatibility).
