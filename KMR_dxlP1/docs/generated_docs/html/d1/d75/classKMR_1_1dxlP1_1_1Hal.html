Detailed Description

Hardware abstraction layer for Dynamixel motors.

The lowest-level element in the library. The Hal class serves primarily as an abstraction layer, providing high-level functions to get the Dynamixel control table addresses by creating a control table.
It also parses the project's motors configuration file.
